What is the difference between non-semantic and semantic tags in HTML?

In HTML, there are both non-semantic tags and semantic tags. I will briefly explain the difference between both. Semantic tags are, put simply descriptive tags. What are two (or more) benefits of using semantic tags?

Two benefits of using semantic tags are: firstly, using semantic tags you have more consistent code. Each programmer will write code slightly differently using non-semantic tags; however, with semantic tags it is streamlined and consistent. Secondly, the use of semantic code is easier to read and maintain. Using non-semantic tags can get crowded and overwhelming to see visually.
