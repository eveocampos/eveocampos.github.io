Images used in HTML and CSS

What are the differences between presenting an image using an HTML tag (such as <img> ) and presenting it as the background of an HTML element (using the CSS property background-image)?

When presenting an image using an HTML tag the <img> tag is used. This is a tag that is a "void element" that cannot have a child or an end tag and uses two attributes, <src> and <alt>, to be useful. The <src> contains an URL that points the image you want to use. The <alt> is the Alternative Text that describes the image, and is displayed when the image cannot display.

  • Example: <img src="https://www.example.com/images/coffeecup.jpg" alt="coffee cup">

When using the <background-image> property it sets the one or more images for an element.

  • Example: background-image: url("paper.gif")

What other CSS properties effect the displays of background images?

There are many other CSS properties that can effect the displays of background images. Two of those are <width> and <height> properties. These represent the height and width of an image in pixels.

  • Example: <img src="images/coffeecup.jpg" alt="Coffee Cup" width="200" height="141">
